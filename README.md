# Ecos de Sangre | Gothic Valentine's Anthology

A dark, romantic web experience designed for Valentine's Day, featuring a curated collection of vampire-themed poetry presented in an immersive gothic atmosphere.

* **Responsive Interface:** Fully optimized for desktop and mobile devices using Bootstrap 5.3.8.

1.  Download the project files.

4.  Click the **"Entrar"** button on the welcome screen to initialize the audio and enter the anthology.
